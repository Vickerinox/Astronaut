# Wallpapers and Backgrounds

In astronaut it's possible to assign images to the top and bottom screens (reffered to ass wallpaper and background respectively). This is primarily done through themes, however the wallpaper (top screen) can have it's imaged overriden by the main settings file.

For these backgrounds you must provide a 256x192 BMP image using either 8-bit, 16-bit (r5g5b5a/x1) or 24-bit color format. Only the 40 byte "BM" header has been tested, so if your background does not work despite these characteristics attempt to re-export it using a program like GIMP.
