# Hinddoor
Hinddoor is a custom firmware made exclusively for DSi consoles. Which is an alternative Nocash's Unlaunch. 
Effectively, it hijacks the console startup procedure while loading the TMD data for the DSi Menu. Unlocking all features of the console for homebrew use.

## Features
* Launch Homebrew and DSiWare software from the DSi SD card and DSi NAND
* A tiny file explorer GUI with touchscreen support
* Background Music
* Selective Autobooting via button combos during reset/startup

## Compatability with Unlaunch
In order to make sure there is not a sea of chaos within the DSi Modding community, the `a+b` button combo is fixed to divert startup to the Hinddoor menu.

## Compatability with slot-1
Currently there is no support for launching the cartridge inserted into slot-1. Instead, it is recommended to autoboot the DSi Menu or a homebrew slot-1 launcher to get this functionality.
